# 洪荒纪元大事记

> 本大事记融合中国古代神话（《山海经》《淮南子》《三五历纪》等）、道教传说、民间信仰以及现代网络文学创作体系，按时间顺序梳理洪荒世界从混沌开辟到末法时代的完整脉络，供小说创作参考。

---

## 一、混沌纪元

### 时间定位

天地未生之前，无光无暗，无始无终，唯有混沌一片。此纪元无时间概念可言，是"无极"之境，先于一切存在。

### 主要事件

混沌之中，三千大道自行运转，孕育出三千混沌魔神。每一尊混沌魔神皆对应一条大道法则，生而有大神通，各执一道，互不相属。混沌无序，魔神各行其是，或沉睡、或争斗、或悟道，无穷岁月中彼此消长。

盘古为三千混沌魔神中最强者，其对应之力为"力之法则"，亦即开天辟地之大道。盘古于混沌中沉睡，感悟开天之机，手持开天神斧，等待时机劈开混沌、再造天地。

### 关键人物

| 人物 | 身份与特征 |
|------|-----------|
| **盘古** | 最强混沌魔神，执力之法则，手持开天神斧，以力证道之第一人 |
| **鸿钧** | 执道之法则，后为合道之人，紫霄宫之主 |
| **杨眉** | 执空间法则，本体为空心杨柳，收容万物，与鸿钧同辈 |
| **罗睺** | 执杀戮法则，魔道之祖，性嗜杀伐，后为魔祖 |
| **乾坤** | 执乾坤法则，主天地阴阳 |
| **阴阳** | 执阴阳法则，主万物生灭 |
| **五行** | 执五行法则，主金木水火土相生相克 |

> 注：三千混沌魔神名号众多，以上仅列在网络文学体系中较为常见者。其余如命运、因果、轮回、造化等法则之魔神，各有其名，体系繁杂，不一而足。

### 起因经过

混沌本无序，三千魔神各执一道，彼此制衡又彼此冲突。盘古悟得开天之机，欲以力证道，开辟新天地。此念一出，便与混沌之"存"相悖——混沌若开，三千魔神所依存之法则根基将被动摇，故开天之举必遭群魔阻挠。

盘古于混沌中积蓄力量，等待最恰当的时机，此等待便不知几万万年。

### 历史含义

混沌纪元是洪荒一切故事的原点。三千大道与三千魔神的存在，奠定了后世一切法则运行的基础。盘古的开天之志，则是混沌走向有序的第一推动力。此纪元的核心命题是：**从无序到有序的第一次尝试**。

## 二、开天大劫

### 时间定位

混沌纪元之末，盘古觉醒，以开天神斧劈开混沌，天地初分。此为洪荒世界之开端，亦为第一量劫。

### 主要事件

盘古觉醒，持开天神斧劈向混沌。一斧之下，混沌破碎，清气上升为天，浊气下沉为地，天地初开。

然而开天之举动摇了三千魔神的根基，三千混沌魔神群起而攻之，欲阻止盘古开辟天地。盘古以力证道，力战三千魔神，斩杀殆尽。魔神陨落，其本源大道散入新开天地，成为洪荒世界法则运行的根基。

大战之后，盘古力竭身殒，但其身躯并未消亡，而是化生万物，成就洪荒世界：

#### 盘古化生详解

| 盘古部位 | 化生之物 | 说明 |
|---------|---------|------|
| **元神三分** | 三清：老子（太清）、元始（玉清）、通天（上清） | 盘古元神一气化三清，各得盘古一分道果 |
| **精血十二份** | 十二祖巫 | 不修元神，肉身无双，各掌一种天地伟力 |
| **左眼** | 太阳星 | 后孕育出帝俊、东皇太一 |
| **右眼** | 太阴星 | 后孕育出常羲、羲和等太阴神祇 |
| **身躯** | 日月星辰、山河大地、草木金石 | 洪荒世界的物质基础 |
| **精血余沥** | 诸多先天大妖、先天神灵 | 散落洪荒各处 |
| **汗珠** | 江河湖海 | 水系之源 |
| **呼吸** | 风云雷电 | 天象之源 |
| **声音** | 雷霆 | 天威之显 |
| **喜怒** | 阴晴寒暑 | 气候之源 |

#### 开天三宝

开天神斧在盘古身殒后一分为三：

| 法宝 | 来源 | 归属 | 威能 |
|------|------|------|------|
| **太极图** | 斧背所化 | 后赐老子 | 平息地水火风，分理阴阳，镇压气运 |
| **盘古幡** | 斧刃所化 | 后赐元始天尊 | 斩裂时空，攻伐第一 |
| **混沌钟（东皇钟）** | 斧柄所化 | 后归东皇太一 | 镇压鸿蒙，逆转时空，防御第一 |

### 关键人物

- **盘古**：开天辟地之主，力证道而身殒，身化万物
- **三千混沌魔神**：阻挠开天，被盘古斩杀殆尽，大道散入洪荒

### 古籍出处

- 《三五历纪》："天地浑沌如鸡子，盘古生其中。万八千岁，天地开辟，阳清为天，阴浊为地。"
- 《五运历年记》："盘古之君，龙首蛇身，嘘为风雨，吹为雷电，开目为昼，闭目为夜。"
- 《绎史》引《五运历年记》："首生盘古，垂死化身：气成风云，声为雷霆，左眼为日，右眼为月，四肢五体为四极五岳，血液为江河，筋脉为地里，肌肉为田土，发髭为星辰，皮毛为草木，齿骨为金石，精髓为珠玉，汗流为雨泽。"

### 历史含义

开天大劫是洪荒世界的创世之战。盘古以一己之力开辟天地、斩杀三千魔神，其牺牲成就了整个洪荒世界的物质与法则基础。此纪元的核心命题是：**以牺牲换取新生，以毁灭缔造创造**。三清与十二祖巫的分立，也为后世巫妖量劫埋下了伏笔——元神与肉身、道法与力道的根本分歧，贯穿洪荒始终。

## 三、凶兽大劫 / 神魔时代

### 时间定位

开天之后，天地初成，先天灵气充沛至极，洪荒尚处蛮荒状态。此纪元为先天神魔化形、凶兽横行之时代。

### 主要事件

天地初开，先天灵气弥漫，万物化形。盘古精血与混沌残余之力交织，催生出大量凶兽。凶兽无灵智，唯知杀戮吞噬，横行洪荒，所过之处生灵涂炭。

与此同时，先天神魔纷纷化形而出。这些先天神魔或为盘古余气所化，或为天地法则自行凝聚而成，生而有大神通，各有灵智。面对凶兽之患，先天神魔或潜修于洞天福地，或游历洪荒斩杀凶兽，各寻机缘。

凶兽大劫并无明确的终结之战，而是随着先天神魔逐渐成长、凶兽被一一斩杀，洪荒逐步从蛮荒走向有序。此过程漫长而零散，无统一战场，亦无主导者。

### 关键人物

- **先天神魔**：鸿钧、杨眉、罗睺、三清、十二祖巫、女娲、伏羲、镇元子、冥河等，皆在此纪元化形或初露头角
- **凶兽**：无灵智，种类繁多，以量取胜，最强者可匹敌大罗金仙

### 起因经过

开天之后，混沌残余之力与盘古精血散落洪荒，催生凶兽。凶兽本能地吞噬灵气与生灵，威胁一切有灵智之存在的生存。先天神魔为自保与修行，逐步清除凶兽，此过程即为"凶兽大劫"。

凶兽大劫的结束标志着洪荒从无序蛮荒进入有序争雄的新阶段——先天神魔清除了最大的外部威胁后，彼此之间的矛盾开始显现，为龙汉初劫埋下伏笔。

### 历史含义

凶兽大劫是洪荒世界的"童年期"。天地初成，秩序未立，一切都在野蛮生长。此纪元的核心命题是：**从蛮荒到文明的过渡**。凶兽代表了混沌残余的无序力量，先天神魔则代表了新秩序的萌芽。凶兽大劫的结束，意味着洪荒世界具备了建立文明秩序的基础。

## 四、龙汉初劫（三族争霸）

### 时间定位

凶兽大劫之后，洪荒先天灵气依然充沛，先天种族繁衍兴盛。此纪元为洪荒第一次大规模种族战争。

### 主要事件

#### 三族崛起

先天三族凭借各自优势，成为洪荒最强大的三股势力：

| 种族 | 首领 | 势力范围 | 特征 |
|------|------|---------|------|
| **龙族** | 祖龙 | 水族、四海 | 掌水之法则，龙族天生神通，呼风唤雨 |
| **凤族** | 元凤 | 飞禽、天空 | 掌火之法则，凤族涅槃重生，焚天煮海 |
| **麒麟族** | 始麒麟 | 走兽、大地 | 掌土之法则，麒麟仁兽，厚德载物 |

三族各据一方，本可相安无事，然而天地气运有限，三族皆欲独占洪荒气运，成为天地主角。

#### 罗睺暗中挑拨

魔祖罗睺趁三族争端初起之际，暗中布局。罗睺以杀戮法则挑动三族仇恨，制造摩擦，推波助澜，使三族矛盾不断升级。罗睺的真实目的是：**以三族大战的血煞之力，祭炼诛仙四剑，成就其以杀证道之大业**。

罗睺布局深远：
- 暗中赐三族首领修炼之法，助其壮大实力，加深野心
- 制造三族之间的流血冲突，不断激化矛盾
- 以分身潜入三族，挑拨离间
- 意图以三族大战的滔天杀戮，冲破天道桎梏

#### 三族大战

三族矛盾终于不可调和，全面战争爆发。大战之惨烈，天地变色，日月无光：

- **第一阶段**：边境冲突，小规模交战，互有胜负
- **第二阶段**：全面开战，三族倾巢而出，洪荒大地沦为战场
- **第三阶段**：决战，三族首领亲上阵，以命相搏

大战最终结果：三族同归于尽，退出洪荒历史舞台。

| 种族 | 结局 |
|------|------|
| **龙族** | 祖龙陨落，残部镇守四海，永世不得出海，龙族气运大损 |
| **凤族** | 元凤陨落，残部栖于不死火山，凤族几乎不再出世 |
| **麒麟族** | 始麒麟陨落，麒麟几乎销声匿迹，偶有出世即为祥瑞 |

#### 鸿钧与罗睺之战

三族大战的真相被鸿钧识破。鸿钧出手，与罗睺展开洪荒第一场巅峰对决：

- 罗睺以诛仙四剑布诛仙剑阵，号称"非四圣不可破"
- 鸿钧以造化玉碟镇压杀气，联合杨眉等人破阵
- 罗睺最终败亡，但其残魂遁入魔界，留下后患
- 诛仙四剑散落洪荒，后为通天教主所得

#### 鸿钧斩三尸成圣

大战之后，鸿钧斩去自身三尸（善尸、恶尸、自我尸），修为臻至圆满，登顶圣人之位，成为洪荒第一位圣人（非六圣之列，为道祖之圣）。

### 关键人物

- **祖龙**：龙族之主，掌水之法则，战败陨落
- **元凤**：凤族之主，掌火之法则，战败陨落
- **始麒麟**：麒麟族之主，掌土之法则，战败陨落
- **罗睺**：魔祖，执杀戮法则，暗中操纵三族大战，最终败于鸿钧
- **鸿钧**：道祖，执道之法则，击败罗睺，斩三尸成圣

### 古籍出处

- 龙汉初劫之名不见于正统古籍，为道教劫运说与网络文学融合之产物
- 道教有"龙汉延康"等劫名，见于《度人经》等道经，但具体内容与网络文学体系差异甚大
- 三族争霸之叙事框架主要源自现代网络文学创作

### 历史含义

龙汉初劫是洪荒第一次因气运争夺而引发的种族大战。此纪元的核心命题是：**气运之争的残酷性**。三族为争夺天地气运而同归于尽，揭示了洪荒世界的根本法则——天地气运有限，种族兴衰有定数，强求者必遭反噬。

罗睺的失败则表明：**以杀证道之路在天道之下终不可行**。鸿钧的成圣，确立了"顺天证道"的正统地位。

三族退出历史舞台后，洪荒权力真空，为巫妖两族的崛起创造了条件。

## 五、紫霄宫讲道

### 时间定位

龙汉初劫之后，鸿钧成圣，于三十三天外紫霄宫开坛讲道。此纪元为洪荒道统确立之始。

### 主要事件

#### 三次讲道

鸿钧于紫霄宫三次讲道，每次讲道皆论大道至理，传修行之法：

| 次数 | 内容 | 要旨 |
|------|------|------|
| **第一次** | 讲大道本源、天地法则 | 阐述三千大道之根本，奠定修行之基 |
| **第二次** | 讲证道之法 | 论述三种证道之路：以力证道、斩三尸证道、功德证道 |
| **第三次** | 讲天道运行、量劫因果 | 揭示量劫之因由，论天道因果轮回之理 |

#### 听道者

紫霄宫讲道，洪荒大神通者纷至沓来。然紫霄宫席位有限，能入宫听道者皆为有大机缘之人：

**紫霄宫三千客**（主要听道者）：

| 人物 | 身份 | 备注 |
|------|------|------|
| **老子** | 盘古元神所化三清之长 | 后为太清圣人 |
| **元始天尊** | 盘古元神所化三清之二 | 后为玉清圣人 |
| **通天教主** | 盘古元神所化三清之三 | 后为上清圣人 |
| **女娲** | 先天神灵，妖族之母 | 后造人成圣 |
| **接引道人** | 西方教主 | 后为阿弥陀佛 |
| **准提道人** | 西方教副教主 | 后为菩提祖师 |
| **镇元子** | 地仙之祖 | 后为万寿山五庄观之主 |
| **鲲鹏** | 妖师 | 后为妖族妖师 |
| **冥河老祖** | 血海之主 | 修血神子之法 |
| **红云** | 先天神灵 | 性情温和，后遭劫陨落 |
| **帝俊** | 太阳星所化 | 后为妖帝 |
| **太一** | 太阳星所化 | 后为东皇 |
| **伏羲** | 先天神灵 | 后为人皇之师 |
| **十二祖巫** | 盘古精血所化 | 不修元神，听道获益有限 |

#### 鸿蒙紫气

鸿钧讲道毕，赐下鸿蒙紫气，为成圣之根基。鸿蒙紫气共七道，代表七圣之位：

| 鸿蒙紫气归属 | 成圣方式 | 备注 |
|-------------|---------|------|
| 老子 | 斩三尸证道 | 人教之主 |
| 元始天尊 | 斩三尸证道 | 阐教之主 |
| 通天教主 | 斩三尸证道 | 截教之主 |
| 女娲 | 功德证道（造人） | 妖族之母 |
| 接引 | 斩三尸证道（立西方教） | 西方教主 |
| 准提 | 斩三尸证道（立西方教） | 西方教副教主 |
| **未知/争议** | — | 有说归于天道，有说归于后土，有说散入洪荒 |

> 注：鸿蒙紫气的第七道归属，在不同网络文学体系中说法不一，是常见的创作悬念。

#### 分宝

鸿钧讲道之后，将开天三宝与诸般先天灵宝分赐众人：

| 法宝 | 赐予 | 备注 |
|------|------|------|
| **太极图** | 老子 | 开天三宝之一 |
| **盘古幡** | 元始天尊 | 开天三宝之一 |
| **混沌钟** | 太一 | 开天三宝之一，后称东皇钟 |
| **诛仙四剑+诛仙阵图** | 通天教主 | 罗睺遗物 |
| **山河社稷图** | 女娲 | 先天灵宝 |
| **十二品莲台** | 接引/准提 | 西方教根基 |
| **造化玉碟** | 鸿钧自留 | 大道之器，后合道所用 |

#### 鸿钧合道

分宝之后，鸿钧以身合天道，成为天道代言人。合道之后：
- 鸿钧不再以独立意志行事，而是代天执道
- 天道运转更加完善，量劫因果有了明确的执行者
- 洪荒世界从"无序争雄"进入"天道主导"的新阶段

### 关键人物

- **鸿钧**：道祖，紫霄宫之主，讲道传法，合道成天道代言人
- **三清**：盘古元神所化，鸿钧记名弟子，后皆成圣
- **女娲**：先天神灵，后造人成圣
- **接引、准提**：西方教双圣，立西方教而证道

### 古籍出处

- 紫霄宫讲道之叙事主要源自《封神演义》中鸿钧老祖的形象，结合道教"道祖"概念扩展而成
- 鸿蒙紫气、分宝等细节为网络文学创作体系的核心设定
- 道教经典中鸿钧之名不见正史，其形象融合了道教"元始天王"等概念

### 历史含义

紫霄宫讲道是洪荒道统确立的里程碑。此纪元的核心命题是：**秩序的建立与道统的传承**。鸿钧以讲道的方式将修行之法传于洪荒众生，确立了"顺天证道"的正统路径，同时也通过鸿蒙紫气预设了六圣之位，为后世量劫埋下伏笔。

鸿钧合道则标志着洪荒从"圣人自治"进入"天道统御"的新阶段——此后一切量劫皆为天道运行之必然，而非偶然。

---

## 六、巫妖量劫（核心纪元）

### 时间定位

紫霄宫讲道之后，巫妖两族崛起，争霸洪荒。此纪元为洪荒第二次量劫，亦是规模最大、影响最深远的一次，直接决定了人族的崛起。

### 主要事件

#### 妖族立天

帝俊与东皇太一为太阳星所化之先天大妖，修为通天，法力无边。二人秉承天命，建立天庭，统御万天：

- **天庭建立**：帝俊为妖帝，太一为东皇，统御周天星斗，号令万妖
- **妖师鲲鹏**：鲲鹏为妖师，创妖文，统一妖族语言文字，制定妖族法度
- **周天星斗大阵**：帝俊以周天星辰为根基，炼制周天星斗大阵，以三百六十五位妖神为阵眼，可聚周天星辰之力，威力无穷
- **妖族兴盛**：万妖来朝，妖族势力遍及洪荒，天庭威压万方
- **羲和嫁帝俊**：太阴星之主羲和嫁于帝俊，生十大金乌

#### 巫族掌地

十二祖巫为盘古精血所化，不修元神，专修肉身，以力证道。十二祖巫各掌一种天地伟力，肉身无双，联手可布十二都天神煞大阵：

| 祖巫 | 所掌之力 | 特征 |
|------|---------|------|
| **帝江** | 空间 | 十二祖巫之首，速度第一 |
| **烛九阴** | 时间 | 掌昼夜更替，时间流速 |
| **蓐收** | 金 | 掌秋收肃杀 |
| **句芒** | 木 | 掌春生万物 |
| **共工** | 水 | 掌江河湖海 |
| **祝融** | 火 | 掌烈焰焚烧 |
| **后土** | 土 | 掌大地厚德，后化轮回 |
| **玄冥** | 冰/雨 | 掌冰雪霜寒 |
| **强良** | 雷 | 掌雷霆之威 |
| **翕兹** | 电 | 掌闪电之力 |
| **奢比尸** | 天气 | 控制气象变化 |
| **天吴** | 风 | 控制风之力量 |

**十二都天神煞大阵**：十二祖巫联手所布之阵，可凝聚盘古真身，重现盘古开天之威。此阵为巫族最强底牌，亦是妖族最为忌惮之力。

巫族不修元神，故无法使用法宝，亦不受精神攻击影响。肉身之强，洪荒无出其右。巫族以大地为根基，掌控洪荒地脉，与妖族的天庭分庭抗礼。

#### 巫妖矛盾

巫妖两族一掌天一掌地，本可各安其位，但矛盾不可调和：
- **根本矛盾**：妖族修元神，巫族不修元神，道途根本对立
- **资源争夺**：洪荒灵地有限，两族势力范围不断冲突
- **种族仇恨**：妖族以万妖为兵，巫族以大巫为将，小规模冲突不断
- **气运之争**：天地气运唯有一份，巫妖必有一争

#### 巫妖大战关键事件

**1. 夸父逐日——金乌杀夸父**

十大金乌为帝俊与羲和之子，顽劣不堪。一日，十日同出，洪荒大地焦灼，生灵涂炭。大巫夸父不忍生灵受难，逐日而行，欲赶走金乌。然金乌以太阳真火焚之，夸父力竭而亡。

> 古籍出处：《山海经·海外北经》："夸父与日逐走，入日；渴，欲得饮，饮于河、渭，河、渭不足，北饮大泽。未至，道渴而死。弃其杖，化为邓林。"

**2. 后羿射日——射杀九只金乌**

夸父之死激怒大巫后羿。后羿以先天灵弓射落九只金乌，仅留一只（后为太阳星君）。帝俊痛失九子，怒不可遏，巫妖仇恨彻底不可调和。

> 古籍出处：《淮南子·本经训》："逮至尧之时，十日并出，焦禾稼，杀草木，而民无所食……羿仰射十日，中其九日，日中九乌皆死，堕其羽翼。"

> 注：在洪荒体系的时间线中，后羿射日被置于巫妖大战期间，而非尧帝时期，这是网络文学对神话时间线的重构。

**3. 共工怒撞不周山——天柱折，天河倒灌**

巫妖大战中，水神共工与火神祝融交战（一说共工与颛顼争帝），共工战败，怒撞不周山。不周山为洪荒天柱，天柱一折，天河之水倒灌洪荒，生灵涂炭，天地倾覆。

> 古籍出处：《淮南子·天文训》："昔者共工与颛顼争为帝，怒而触不周之山，天柱折，地维绝。天倾西北，故日月星辰移焉；地不满东南，故水潦尘埃归焉。"

**4. 女娲炼石补天——斩神龟立四极**

不周山折断后，天河倒灌，洪荒面临灭世之灾。女娲不忍生灵涂炭，炼五色石以补苍天，斩巨龟之足以立四极，聚芦灰以止淫水。天地重归安定。

女娲补天之功，为洪荒第一功德，亦为其后造人成圣积累了无上功德。

> 古籍出处：《淮南子·览冥训》："于是女娲炼五色石以补苍天，断鳌足以立四极，杀黑龙以济冀州，积芦灰以止淫水。"

**5. 后土化轮回——舍身开辟六道轮回**

巫妖大战中，生灵死伤无数，亡魂无处安息，化为孤魂野鬼，洪荒怨气冲天。祖巫后土不忍亡魂受苦，舍身化六道轮回，使亡魂得以转世投胎。

后土化轮回为洪荒最伟大的牺牲之一：
- 六道轮回建立，生死有序，亡魂有归
- 后土肉身消亡，仅余残余意志化为后土皇地祇
- 巫族失去一位祖巫，实力大损
- 此举功德无量，但后土未能成圣（一说因巫族不修元神之故）

> 注：后土化轮回之说不见于正统古籍，为网络文学体系的重要创作。古籍中后土为社稷之神，《礼记·祭法》："共工氏之霸九州也，其子曰后土，能平九州，故祀以为社。"

**6. 屠巫剑——东皇太一以三亿人族精血祭炼**

巫妖大战后期，东皇太一以三亿人族精血祭炼屠巫剑，此剑专克巫族肉身，一剑可斩祖巫。此为巫妖大战中最残忍之举，亦是人族在洪荒中遭受的第一次灭族之灾。

屠巫剑的存在，使得人族与巫妖两族皆有血仇，为人族日后的独立崛起埋下了种子。

> 注：屠巫剑为网络文学创作，不见于古籍。

**7. 巫妖决战——两族同归于尽**

巫妖大战最终决战：
- 十二祖巫布十二都天神煞大阵，凝聚盘古真身
- 帝俊、东皇太一率周天星斗大阵迎击
- 两族倾尽所有，洪荒天地崩碎
- 帝俊陨落，东皇太一陨落
- 祖巫大部陨落，仅余零星残部

#### 大战结局

| 势力 | 结局 |
|------|------|
| **妖族天庭** | 崩塌，帝俊、太一俱殒，妖族四散，天庭空虚 |
| **巫族** | 几乎灭绝，祖巫仅余后土残魂（化轮回）等，大巫残部退入幽冥 |
| **洪荒天地** | 彻底残破，灵气大减，天地格局剧变 |
| **人族** | 遭屠巫剑之祸，三亿人族惨死，但亦因此脱离巫妖附庸，走向独立 |

### 关键人物

- **帝俊**：妖帝，太阳星所化，建立天庭，陨于决战
- **东皇太一**：东皇，太阳星所化，持混沌钟，陨于决战
- **鲲鹏**：妖师，创妖文，战后遁走
- **十二祖巫**：盘古精血所化，掌天地伟力，大部陨落
- **后土**：祖巫，舍身化六道轮回
- **夸父**：大巫，逐日而亡
- **后羿**：大巫，射杀九金乌
- **共工**：祖巫，怒撞不周山
- **女娲**：补天救世，后造人成圣

### 古籍出处

- 巫妖量劫之整体框架为网络文学创作，融合了多处古籍元素
- 《山海经》：夸父逐日、刑天舞干戚等
- 《淮南子》：后羿射日、女娲补天、共工触山
- 《楚辞·九歌》：东皇太一之名
- 《三五历纪》：盘古化生万物

### 历史含义

巫妖量劫是洪荒世界最核心的纪元，其影响深远：
1. **天地格局重塑**：巫妖两族同归于尽，天庭空虚，为后世封神量劫埋下伏笔
2. **人族命运转折**：人族在巫妖大战中遭受屠戮，但亦因此独立，成为天地新主角
3. **六道轮回建立**：后土化轮回，使生死有序，为后世因果轮回奠定基础
4. **天地残破**：洪荒灵气大减，先天灵气逐渐转化为后天灵气，修士境界上限降低

此纪元的核心命题是：**旧秩序的毁灭与新秩序的孕育**。巫妖两族的覆灭，为人族的崛起扫清了道路。

---

## 七、人族崛起时代

### 时间定位

巫妖量劫之后，天地残破，巫妖退出历史舞台，人族被天地定为新主角。此纪元从女娲造人开始，历经三皇五帝，直至大禹治水、夏朝建立。

### 主要事件

#### 女娲造人

巫妖大战后，洪荒生灵凋零，天地需新主角承载气运。女娲感悟天机，以黄土抟人，造出人族：

- **造人过程**：女娲以九天息壤和黄土，抟土造人。先以手精心捏制，后以藤条沾泥甩洒，故人有贵贱之分（此说见《风俗通义》）
- **功德成圣**：女娲造人，功德无量，以此功德证道成圣，为洪荒第六位圣人
- **人族定位**：天地降下法旨，定人族为洪荒新主角，人族气运从此与天地相连
- **人族特征**：人族先天不足，无妖族之体魄、无巫族之肉身、无先天神灵之法力，但人族有悟性、有创造力、有繁衍之力，且为天地主角，气运加身

> 古籍出处：
> - 《风俗通义》："俗说天地开辟，未有人民，女娲抟黄土作人。剧务，力不暇供，乃引绳于泥中，举以为人。故富贵者，黄土人也；贫贱凡庸者，絙人也。"
> - 《太平御览》引《风俗通义》
> - 《山海经·大荒西经》："有神十人，名曰女娲之肠，化为神，处栗广之野，横道而处。"

#### 人族三祖

人族初生，茹毛饮血，与禽兽无异。三祖出世，教化人族，使人族从蒙昧走向文明：

| 三祖 | 贡献 | 意义 |
|------|------|------|
| **燧人氏** | 钻木取火 | 终结茹毛饮血，人族始有熟食，夜间有光，驱赶野兽 |
| **有巢氏** | 建造巢居 | 让人族有了栖身之所，不再露宿荒野，减少野兽侵害 |
| **缁衣氏** | 织布制衣 | 御寒保暖，使人族不再以兽皮裹身，文明之始 |

三祖之贡献看似平凡，实为人族立足洪荒之根基。火、居、衣三者，使人族从"与禽兽同"变为"与禽兽异"，文明由此萌芽。

> 注：三祖之说散见于诸子百家，具体名号在不同典籍中有所差异。

#### 三皇治世

**天皇——伏羲**

伏羲为先天神灵转世（一说与女娲为兄妹），降生人族，教化万民：
- **创八卦**：仰观天象，俯察地理，创先天八卦，为人族开启智慧之门
- **定婚姻**：制定婚嫁之礼，使人族有序繁衍，不再乱婚
- **制渔网**：教人结网捕鱼，使人族有了稳定的食物来源
- **造琴瑟**：制乐器，通天地之音，人族始有礼乐

> 古籍出处：《周易·系辞下》："古者包牺氏之王天下也，仰则观象于天，俯则观法于地，观鸟兽之文与地之宜，近取诸身，远取诸物，于是始作八卦。"

**地皇——神农**

神农为人族之圣者，以仁德治天下：
- **尝百草**：亲尝百草，一日遇七十二毒，以茶解之，为人族开辟医药之道
- **兴农耕**：教人种植五谷，制作农具，使人族从采集狩猎进入农耕文明
- **立市廛**：建立集市，使人族互通有无，商业萌芽

> 古籍出处：《淮南子·修务训》："神农尝百草之滋味，水泉之甘苦，令民知所避就。当此之时，一日而遇七十毒。"

**人皇——轩辕**

轩辕黄帝为人族最伟大的帝王，统一人族，奠定华夏根基：
- **战蚩尤**：涿鹿之战击败蚩尤，统一人族各部
- **定九州**：划分九州，建立人族疆域
- **制器物**：造舟车、制衣裳、定历法、创文字（命仓颉造字）
- **兴制度**：建立官制、军制、礼制，人族从此有国家雏形

> 古籍出处：《史记·五帝本纪》

#### 涿鹿之战

涿鹿之战是人族崛起时代最关键的战争，决定了人族的命运：

**前奏——阪泉之战**

轩辕与炎帝（神农氏后裔）争人族共主之位，战于阪泉。三战之后，炎帝败北，与轩辕合并，形成炎黄联盟。此为人族内部统一之战。

> 古籍出处：《史记·五帝本纪》："轩辕乃修德振兵……以与炎帝战于阪泉之野。三战，然后得其志。"

**涿鹿之战**

蚩尤为九黎之主，大巫转世（一说为巫族残部），铜头铁额，力大无穷，有兄弟八十一人，皆兽身人语，食沙石，造兵杖刀戟大弩，威震天下。

战争经过：
1. 蚩尤率九黎部北进，与炎黄联盟对峙
2. 蚩尤作大雾，黄帝造指南车破之
3. 蚩尤请风伯雨师，纵大风雨，黄帝请天女魃止雨
4. 最终黄帝以轩辕剑斩蚩尤，九黎部溃散

**轩辕剑**：太上老君以首山之铜重铸屠巫剑而成。屠巫剑以三亿人族精血祭炼，杀孽深重，老君以人族气运洗涤其杀气，重铸为轩辕剑，为人族圣器，代表天道对人族主角地位的认可。

> 古籍出处：
> - 《史记·五帝本纪》："蚩尤作乱，不用帝命。于是黄帝乃征师诸侯，与蚩尤战于涿鹿之野，遂禽杀蚩尤。"
> - 《逸周书·尝麦解》："蚩尤乃逐帝，争于涿鹿之阿，九隅无遗。赤帝大慑，乃说于黄帝，执蚩尤，杀之于中冀。"
> - 《山海经·大荒北经》："蚩尤作兵伐黄帝，黄帝乃令应龙攻之冀州之野。应龙畜水。蚩尤请风伯雨师，纵大风雨。黄帝乃下天女曰魃，雨止，遂杀蚩尤。"

> 注：轩辕剑为屠巫剑重铸之说为网络文学创作，不见于古籍。

#### 五帝治世

**颛顼——绝地天通**

颛顼为黄帝之孙，其最大功绩为"绝地天通"：
- 断绝天地之间的通道，使人神分治
- 天神不得随意下界，人族不得随意登天
- 此举标志着人族正式独立于神族之外，人族自有其道

> 古籍出处：《尚书·吕刑》："乃命重黎，绝地天通，罔有降格。"《国语·楚语下》详述此事。

**帝喾——正四时、施仁德**

帝喾为颛顼之侄，仁德之君：
- 制定四时节令，使人族农事有序
- 施仁政，天下归心

**尧——制定历法、禅让制**

尧帝为帝喾之子，圣德之君：
- 制定完善历法，以三百六十六日为一年
- 创禅让制，不传子而传贤
- 派羿射日（此为另一版本的后羿射日传说，与巫妖量劫中的后羿射日不同）

> 古籍出处：《史记·五帝本纪》

**舜——以德治国、选大禹**

舜帝以孝德闻名，受尧禅让而登帝位：
- 以德治国，天下太平
- 选大禹治水，解决洪患
- 禅让于禹

**大禹——治水、铸九鼎**

大禹为人族治世之终章：
- **治水**：大禹治水十三年，三过家门而不入，疏导九河，平息水患
- **铸九鼎**：收九州之金，铸九鼎，一鼎代表一州，九鼎为人族气运之器，亦为王权象征
- **建夏朝**：大禹之子启建立夏朝，人族从禅让制进入世袭制

> 古籍出处：
> - 《尚书·禹贡》
> - 《史记·夏本纪》
> - 《左传》："昔夏之方有德也，远方图物，贡金九牧，铸鼎象物。"

### 关键人物

- **女娲**：造人之圣，人族之母
- **燧人氏、有巢氏、缁衣氏**：人族三祖，文明奠基者
- **伏羲**：天皇，开人族智慧
- **神农**：地皇，兴农耕医药
- **轩辕黄帝**：人皇，统一人族
- **蚩尤**：九黎之主，大巫转世
- **颛顼**：绝地天通，人神分治
- **大禹**：治水铸鼎，人族治世之终

### 历史含义

人族崛起时代是洪荒世界从"神族主导"转向"人族主导"的关键纪元。此纪元的核心命题是：**人族如何从洪荒最弱小的种族，成长为天地主角**。

关键线索：
1. **女娲造人**确立了人族的天地地位
2. **三祖教化**使人族具备了生存能力
3. **三皇治世**使人族建立了文明秩序
4. **涿鹿之战**统一了人族，轩辕剑象征天道对人族的认可
5. **绝地天通**使人族独立于神族
6. **大禹治水铸九鼎**标志着人族完全自立

从巫妖量劫中的屠巫剑之祸，到大禹铸九鼎立夏朝，人族完成了从"附庸"到"主角"的蜕变。

## 八、封神量劫

### 时间定位

人族崛起之后，天地秩序尚未完善。天庭虽立，但神位空虚，三界秩序不稳。鸿钧降下法旨，开启封神大业，以战争方式填补天庭神位。

### 主要事件

#### 封神起因

- 天庭建立后，昊天上帝（一说为鸿钧道童所化）为天庭之主，但手下无兵无将，形同光杆司令
- 昊天请命于鸿钧，请求充实天庭
- 鸿钧降下法旨：三教（阐教、截教、人教）门下弟子犯杀劫者，皆入封神榜，死后为天庭正神
- 封神榜共三百六十五位正神，需以量劫填之

#### 阐教 vs 截教

| 阵营 | 教主 | 理念 | 主要弟子 |
|------|------|------|---------|
| **阐教** | 元始天尊 | 顺应天命，门下皆为根行深厚之人 | 十二金仙（广成子、赤精子、太乙真人等） |
| **截教** | 通天教主 | 有教无类，广收门徒，不问出身 | 万仙来朝，弟子无数（多宝道人、金灵圣母、无当圣母、龟灵圣母等） |

两教理念根本对立：阐教重"根行"，截教重"有教无类"。封神之战表面上是商周更替，实则是阐截两教的道统之争。

#### 商周更替（人间战场）

封神之战以商周更替为人间投影：
- **商纣王**：暴虐无道，亵渎女娲，截教支持商朝
- **周武王**：仁德之君，顺应天命，阐教支持周朝
- **姜子牙**：元始天尊门下，代天封神，执封神榜与打神鞭

#### 关键大战

**1. 诛仙阵**

通天教主以诛仙四剑布诛仙剑阵，号称"非四圣不可破"。老子、元始天尊、接引、准提四圣联手破阵，诛仙四剑被夺。

**2. 万仙阵**

通天教主倾截教全力，布万仙大阵。阐教联合西方教双圣，再次破阵。截教万仙凋零，截教从此一蹶不振。

**3. 其他重要战役**

- 十绝阵、九曲黄河阵、瘟癀阵等
- 诸多截教弟子战死上榜
- 申公豹以"道友请留步"坑杀无数同门

#### 封神结局

- 三百六十五路正神归位，天庭兵将齐备
- 截教覆灭，通天教主被鸿钧带回紫霄宫禁闭
- 阐教亦损失惨重，十二金仙多人犯杀劫
- 西方教趁势做大，收走截教大量弟子（如多宝道人化为如来）
- 人间商亡周立，周朝八百年基业

### 关键人物

- **姜子牙**：封神执行者，执封神榜与打神鞭
- **通天教主**：截教之主，布诛仙、万仙二阵
- **老子、元始天尊**：阐教一方，破阵主力
- **接引、准提**：西方教双圣，趁势收人
- **纣王**：商朝末代帝王，暴虐亡国
- **周武王**：周朝开国之君
- **哪吒、杨戬、雷震子**：阐教三代弟子中的佼佼者

### 古籍出处

- 《封神演义》（明·许仲琳）：封神量劫的主要文本来源
- 《武王伐纣平话》：封神故事的前身
- 《史记·殷本纪》：商纣王的历史记载

### 历史含义

封神量劫的核心命题是：**秩序的最终确立**。通过封神之战，天庭获得了完整的神官体系，三界秩序从此稳固。但封神之战也是一场残酷的清洗——截教的覆灭，意味着"有教无类"的理念在洪荒世界中失败，"顺应天命"成为唯一的正统。

对人族而言，封神量劫意味着人间的王朝更替从此受天道制约，人族的命运不再完全由自己掌握，而是被纳入三界秩序之中。

---

## 九、西游量劫

### 时间定位

封神量劫之后，三界秩序初立，但佛道气运尚未平衡。西游量劫为佛道两教气运博弈之产物，亦是灵山崛起之关键。

### 主要事件

#### 佛道博弈

封神之战中，西方教（后改称佛教/灵山）趁势做大，收走大量截教弟子。佛门气运大涨，与道门分庭抗礼。然而佛门需将南赡部洲纳入势力范围，方能彻底奠定格局。

如来佛祖（多宝道人化胡为佛）与天庭、道门达成协议：以取经之业，将佛门传入东土。此为西游量劫的根本起因——表面上是唐僧取经，实质上是佛道两教的气运分配。

#### 取经团队

| 成员 | 真实身份 | 角色 |
|------|---------|------|
| **唐僧** | 金蝉子转世（如来二弟子） | 取经核心，十世修行的好人 |
| **孙悟空** | 女娲补天遗石化生 | 斗战胜佛，降妖除魔 |
| **猪八戒** | 天蓬元帅下凡 | 净坛使者 |
| **沙悟净** | 卷帘大将下凡 | 金身罗汉 |
| **白龙马** | 西海龙王三太子 | 八部天龙 |

#### 取经之路

取经之路九九八十一难，表面上是妖魔阻路，实质上多为佛道两教安排的"考核"：
- 有背景的妖怪多为神仙坐骑、童子下凡，最终被主人收回
- 无背景的妖怪则被孙悟空打死
- 每一难都是佛道两教气运博弈的缩影

#### 西游结局

- 唐僧取回真经，佛门传入东土
- 灵山崛起，佛道两教气运达成平衡
- 三界格局定型：天庭管行政，道门管修行，佛门管教化
- 取经团队各有封赏

### 关键人物

- **如来佛祖**：灵山之主，策划取经
- **观音菩萨**：取经执行者，安排八十一难
- **唐僧**：取经核心
- **孙悟空**：齐天大圣，斗战胜佛
- **玉帝**：天庭之主，默许取经
- **太上老君**：道门之主，暗中布局

### 古籍出处

- 《西游记》（明·吴承恩）：西游量劫的主要文本来源
- 《大唐西域记》（唐·玄奘）：历史原型
- 《慈恩传》：玄奘传记

### 历史含义

西游量劫的核心命题是：**三界格局的最终定型**。封神量劫确立了天庭的行政秩序，西游量劫则确立了佛道两教的气运平衡。此后三界各安其位，再无大规模量劫。

对人族而言，西游量劫意味着佛道两教都开始在人间传播，人族的信仰与精神世界从此被纳入三界秩序。人族虽为天地主角，但在精神层面仍受神佛引导。

## 十、末法时代

### 时间定位

西游量劫之后，天地灵气持续衰减，仙道逐渐不显，神佛隐退。从上古至中古至近古，灵气愈稀，修行愈难，直至仙道断绝，进入末法时代。

### 主要事件

#### 灵气衰退

洪荒天地历经数次量劫，灵气不断损耗：
- 开天大劫：先天灵气充沛
- 巫妖量劫：先天灵气大减，后天灵气渐增
- 封神量劫：灵气进一步衰减
- 西游量劫后：灵气稀薄，大能纷纷隐退

灵气衰退导致：
- 修士境界上限不断降低（从圣人→大罗→金仙→天仙→地仙→人仙→凡人）
- 修行难度不断增加
- 天地间灵物逐渐消失
- 仙凡之隔越来越大

#### 诸天隐退

末法时代来临，大能纷纷隐退：
- 圣人闭关不出
- 大罗金仙隐于洞天福地
- 天庭封闭天门，不再接纳新仙
- 灵山佛国亦渐隐于世

#### 人道巅峰

仙道衰退的同时，人道达到巅峰：
- 人族不再依赖仙道，转而发展自身的文明与科技
- 儒释道三教并立，人族精神世界丰富
- 王朝更替，文明传承
- 最终进入现代社会，科技昌明，牛鬼蛇神不存人间

#### 现代社会

末法时代的最终形态：
- 天地灵气几近于无
- 修行之法断绝
- 神仙传说成为故事
- 人族以科技之力掌控世界
- 然天地大道仍在，量劫之机未绝——有说法认为末法时代终将结束，新纪元或将开启

### 关键人物

末法时代无特定关键人物，人族整体为主角。若论影响深远者：
- 历代帝王：推动人族文明发展
- 诸子百家：奠定人族思想根基
- 历代科学家：推动科技进步

### 古籍出处

- 末法时代之说源于佛教"正法、像法、末法"三时说
- 道教亦有类似观点，认为天地有"成住坏空"之循环
- 《法灭尽经》：描述末法时代佛法衰灭之象

### 历史含义

末法时代的核心命题是：**人族的独立与自足**。当仙道隐退、神佛不显，人族必须依靠自身力量生存与发展。这是人族从"天地主角"到"世界主人"的最终蜕变——不再依赖任何超自然力量，以自身的智慧与勇气面对一切。

末法时代亦是洪荒纪元的终章。从混沌开辟到末法降临，洪荒世界走完了从"无序"到"有序"、从"神治"到"人治"的完整历程。
