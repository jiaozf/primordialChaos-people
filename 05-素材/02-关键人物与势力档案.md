# 关键人物与势力档案

## 一、先天圣人体系

### 1. 鸿钧道祖
- 身份：天道代言人，紫霄宫之主
- 修为：合道境（身合天道）
- 法宝：造化玉碟（残）、乾坤鼎
- 主要事迹：紫霄宫三次讲道、分发鸿蒙紫气、合天道
- 出处：《封神演义》首次出现，网文体系完善
- 小说定位：天道秩序的维护者，量劫的幕后推手

### 2. 太清老子（道德天尊）
- 身份：三清之长，人教教主
- 修为：混元大罗金仙（圣人）
- 法宝：太极图、天地玄黄玲珑宝塔、扁拐
- 性格：清静无为，不争不抢
- 主要事迹：立人教、化胡为佛、一气化三清
- 出处：《道德经》《封神演义》
- 小说定位：人族名义上的庇护者，但态度消极

### 3. 玉清元始天尊
- 身份：三清之二，阐教教主
- 修为：混元大罗金仙（圣人）
- 法宝：盘古幡、三宝玉如意、诸天庆云
- 性格：高傲，重根脚门第，心狠手辣
- 主要事迹：立阐教、封神之战主导者
- 出处：《封神演义》
- 小说定位：旧秩序的维护者，人族崛起的潜在阻碍

### 4. 上清通天教主（灵宝天尊）
- 身份：三清之三，截教教主
- 修为：混元大罗金仙（圣人）
- 法宝：诛仙四剑+阵图、青萍剑
- 性格：性情刚烈，护短，重情义
- 主要事迹：立截教、万仙来朝、诛仙阵
- 出处：《封神演义》
- 小说定位：有教无类的代表，但最终败于天道大势

### 5. 女娲娘娘
- 身份：万灵之宗，妖教教主，人族圣母
- 修为：混元大罗金仙（圣人，功德成圣）
- 法宝：山河社稷图、红绣球、招妖幡、宝莲灯
- 性格：慈悲但护短，对人族有特殊感情
- 主要事迹：抟土造人、炼石补天
- 出处：《风俗通义》《淮南子》《山海经》
- 小说定位：人族圣母，但受天道约束不能过度干预

### 6. 接引道人（阿弥陀佛）
- 身份：西方教大教主
- 修为：混元大罗金仙（圣人，大宏愿成圣）
- 法宝：十二品功德金莲、青莲宝色旗
- 性格：不争故不败
- 出处：《封神演义》
- 小说定位：西方势力代表，与人族有间接关系

### 7. 准提道人（菩提祖师）
- 身份：西方教二教主
- 修为：混元大罗金仙（圣人，大宏愿成圣）
- 法宝：七宝妙树、加持神杵
- 性格：四处奔波度化有缘人
- 出处：《封神演义》《西游记》
- 小说定位：最活跃的圣人之一，善于钻营

## 二、妖族核心

### 1. 帝俊
- 身份：妖族天帝
- 法宝：河图洛书、屠巫剑（部分版本）
- 主要事迹：建立妖族天庭、统领万妖
- 结局：巫妖大战中陨落

### 2. 东皇太一
- 身份：妖族至尊
- 法宝：混沌钟（东皇钟）、屠巫剑
- 主要事迹：以混沌钟镇压气运、炼制屠巫剑
- 结局：与祖巫同归于尽
- 出处：《楚辞·九歌·东皇太一》

### 3. 妖师鲲鹏
- 身份：妖族军师
- 法宝：暂无固定（偷取河图洛书）
- 主要事迹：创妖文、助帝俊统一妖族
- 结局：巫妖大战中偷取河图洛书逃走

### 4. 十大妖圣
- 计蒙、英招、白泽、飞诞、飞廉、九婴、呲铁、商羊、钦原、鬼车
- 巫妖大战中死伤惨重

## 三、巫族核心（十二祖巫）

### 1. 帝江（空间祖巫）
- 法则：空间、极速
- 形象：黄囊赤身，六足四翼，面目混沌
- 出处：《山海经·西山经》《庄子·应帝王》
- 结局：巫妖大战中陨落

### 2. 烛九阴（时间祖巫）
- 法则：时间、昼夜
- 形象：人面蛇身，全身赤红
- 出处：《山海经·大荒北经》
- 能力：睁眼为昼，闭眼为夜

### 3. 共工（水之祖巫）
- 法则：水系、洪流
- 形象：蟒头人身，身披黑鳞
- 出处：《山海经·海内经》
- 主要事迹：怒撞不周山，天柱折
- 结局：与祝融大战后撞不周山

### 4. 祝融（火之祖巫）
- 法则：火焰、焚天
- 形象：兽头人身，身披红鳞
- 出处：《山海经·海外南经》
- 结局：与共工大战同归于尽

### 5. 后土（土之祖巫）
- 法则：大地、山川、轮回
- 形象：人身蛇尾，背后七手
- 出处：《山海经·海内经》
- 主要事迹：舍身化六道轮回
- 小说定位：巫族中最慈悲者，为人族和众生开辟轮回

### 6. 蓐收（金之祖巫）
- 法则：金行、杀伐
- 出处：《山海经·西山经》

### 7. 句芒（木之祖巫）
- 法则：木系、生机
- 出处：《山海经·海外东经》

### 8. 玄冥（冰/雨之祖巫）
- 法则：寒冰、雨雪

### 9. 强良（雷之祖巫）
- 法则：雷霆
- 出处：《山海经·大荒北经》

### 10. 天吴（风之祖巫）
- 法则：风系
- 形象：八首人面，虎身十尾

### 11. 翕兹（电之祖巫）
- 法则：雷电
- 出处：《山海经》

### 12. 奢比尸（天气之祖巫）
- 法则：天气变化
- 出处：《山海经·海外东经》

### 巫族核心阵法：十二都天神煞大阵
- 十二祖巫齐聚可凝聚盘古真身
- 威力可开天辟地，圣人之下无敌

## 四、人族核心人物

### 1. 人族三祖
- 燧人氏（火祖）：钻木取火
- 有巢氏（居祖）：建造巢居
- 缁衣氏（衣祖）：织布制衣

### 2. 三皇
- 伏羲（天皇）：创八卦、定婚姻
- 神农（地皇/炎帝）：尝百草、兴农耕
- 轩辕（人皇/黄帝）：战蚩尤、定九州

### 3. 五帝
- 少昊：金德镇世
- 颛顼：绝地天通
- 帝喾：正四时
- 尧：禅让制
- 舜：以德治国

### 4. 其他关键人物
- 大禹：治水、铸九鼎
- 后羿：射日
- 夸父：逐日
- 蚩尤：九黎之主，大巫转世

## 五、其他重要势力

### 2. 凤族
- 元凤后裔，栖于不死火山

### 3. 麒麟族
- 始麒麟后裔，几乎销声匿迹

### 4. 冥河老祖
- 血海之主

- 修阿修罗教

### 5. 镇元子
- 地仙之祖
- 法宝：地书
