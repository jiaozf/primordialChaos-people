# 卷4：人道初立 — 完整章节细纲

> 总章数：70章
> 预计字数：20-25万字（每章2500-3000字）
> 节奏模板：波浪递进型（爆发→低谷→更大爆发→低谷→终极爆发）
> 核心主题：对抗人教阐教，确立人道，人族独立自强

---

## 节奏总览

```
第一幕：新的敌人（第1-22章）    — 建立期，人教阐教介入
第二幕：信仰之争（第23-48章）   — 对抗期，人道理念传播
第三幕：人道确立（第49-70章）   — 高潮期，人族击败阐教弟子
```

## 第一幕：新的敌人（第1-22章）

### 三章闭环 #1：巫妖大战结束（第1-3章）

#### 第1章：大战终结

**章节信息**：
- 字数：2500-3000字
- 节奏定位：冲突引入
- 核心任务：巫妖大战结束，人族迎来曙光

**剧情大纲**：

```
【开篇】
巫妖大战终于结束。
巫族退守北方，妖族退守天庭。
洪荒大地满目疮痍，但人族……活下来了。

【中段】
陈玄站在人道宗的城墙上，看着远方的天际。
那里，曾经是巫妖大战的战场。
现在，只剩下废墟和焦土。

【高潮】
玄黄老人的声音响起：
"巫妖大战……结束了。"
"人族……终于熬过来了。"
"但……这只是开始。"
"什么意思？"
"巫妖两败俱伤，天地主角之位空缺。"
"人族……有机会成为新的天地主角。"
```

**爽点设计**：
- 中爽点：巫妖大战结束
- 章末钩子：人族有机会成为天地主角

**场景渲染**（古风意境渲染师·战场框架）：
```
【战后痕迹】
洪荒大地满目疮痍。
曾经的山河，现在只剩下废墟和焦土。
战场上，散落着巫族和妖族的尸体。
空气中弥漫着浓烈的血腥味。

【旁观者视角】
陈玄站在废墟中，看着那些死去的生灵。
他的心中，没有喜悦，只有沉重。
"这就是量劫……"
"每一次量劫，都会消耗大量天地气运。"
"而这些气运……最终被天道吸收。"

【意境升华】
陈玄抬头，看着天空。
天空中，隐约有一道金色的光芒。
那是天道的光芒。
它在收割着量劫产生的气运。
```
