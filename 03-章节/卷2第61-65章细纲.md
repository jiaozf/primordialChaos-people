### 三、剧情大纲

#### 开篇（约800字）

**场景渲染**（古风意境渲染师·战场框架）：

```
【战后痕迹】
山谷中，到处是昨夜战斗的痕迹。折断的骨矛插在泥地里，巫族的图腾旗帜倒在路边，上面沾满了泥浆与血迹。

【环境与追击的关联】
晨雾弥漫，能见度极低。但巫族溃兵的踪迹清晰可辨——被踩踏的草丛、丢弃的辎重、地上拖行的血痕。

【旁观者视角】
几个躲在山洞中的散修，远远看到人道宗的追兵从雾中现身，整齐的队列、锋利的兵器、坚定的眼神。他们不敢相信，人族竟然在追着巫族打。

【溃败的尺度感】
巫族大军来时一万精锐，此刻溃退的队伍稀稀拉拉，不足三千。大巫的战旗已经不见踪影，只剩下几面残破的小旗在风中抖动。
```

**剧情内容**：
- 苍梧叛变被平息后，巫族大军失去内应，又被陈玄引入陷阱，溃不成军
- 陈玄当机立断，趁巫族立足未稳，率人道宗精锐追击
- 黎明时分，追兵在山谷中追上巫族溃兵的后队
- 小石率前锋营截住巫族退路，巫族进退两难

**剧情内容**：

#### 高潮（约700字）

**文笔润色要点**（文笔润色师·打斗润色）：

```
❌ 叙述型：陈玄击退了大巫残部，巫族溃兵四散而逃。

✅ 展示型：

陈玄没有停步。
他踏出一步，脚下的碎石被灵压震成齑粉。
第二步，骨盾上的纹路开始龟裂。
第三步——
剑光一闪。
骨盾从正中裂成两半，切面光滑如镜。
大巫瞳孔骤缩，后退三步，喉间涌出一口黑血。
"天仙道果……你竟然……"

```

**剧情内容**：
- 陈玄以压倒性实力击溃大巫残部，但选择放走一人传信

- 人道宗追击战大获全胜：巫族一万精锐折损过半，五位大巫一死三伤
- 陈玄下令停止追击——穷寇莫追，北方是巫族腹地，深入太危险
- **伏笔**：巫族残部北撤，但北方巫族主力尚存，这次失败只会让巫族更加愤怒

#### 结尾（约500字）

正午，阳光驱散了山谷中最后的雾气。陈玄站在山口高处，俯瞰着战场上散落的巫族旗帜和骨器。

小石清点完战果，快步走来："宗主，此战巫族折损七千余，大巫一死三伤。我方阵亡十一人，重伤二十三人。"

陈玄沉默片刻。十一人。每一个都是跟随他出生入死的弟子，每一个身后都有一个家庭。

"记下他们的名字，"陈玄低声说道，"抚恤按双倍发放。他们的家人，人道宗养。"

"是。"小石的声音也有些沙哑。

陈玄望向北方。远处的山峦连绵不绝，云雾缭绕。那是巫族的领地，也是更大的危险所在。

"这次追击到此为止。"陈玄转身，"回去，我们还有更重要的事要做。"

**伏笔F071埋设**：巫族残部北撤，但损失惨重只会让巫族更加仇视人族。北方巫族主力尚存，这次失败不是结束，而是更大冲突的前奏。承接伏笔F069。

```
【空间定位】
人道宗大殿。殿门大开，秋风吹入，烛火摇曳。

【人物分布】
陈玄坐在主位，面色平静。小石站在左侧，手持一卷玉简——那是苍梧叛变的全部证据。弟子们分列两侧，目光如刀。

【被告姿态】
苍梧跪在殿中央。他的双手被铁链锁住，衣衫破旧，面色灰败。三天前被抓获时，他还试图反抗，被小石一拳打落了两颗牙。

【气氛渲染】
大殿中安静得可怕。数百双眼睛盯着苍梧，目光中满是愤怒。苍梧打开城门放巫族进来的那一夜，有多少同门死在巫族手下？

**剧情内容**：
- 追击战结束后第三日，陈玄在大殿公开审判苍梧

- 弟子们群情激愤，要求处死苍梧
- 苍梧跪在殿中，面色灰败，但眼中仍有一丝不甘

轻视：苍梧以为陈玄会杀他，甚至做好了赴死的准备
反转：陈玄不杀他，而是将他驱逐——比死亡更残酷的惩罚是让他活着承受众叛亲离
```

**剧情内容**：
- 小石当众宣读苍梧的罪状：与巫族暗中勾结、散布谣言动摇人心、叛变打开城门

- 弟子们齐声高呼："处死叛徒！"

- "我不杀你。"
- 大殿中一片哗然

#### 高潮（约700字）

**文笔润色要点**（文笔润色师·心理润色）：

```
❌ 叙述型：陈玄决定不杀苍梧，而是将他驱逐出人道宗。

✅ 展示型：
"我不杀你。"
这四个字落在殿中，比刀剑更重。
苍梧猛地抬头，眼中满是不可置信。
弟子们也愣住了——不杀？
陈玄的声音平静如水："杀了你，太便宜你了。"
他俯下身，与苍梧平视："你背叛人族，是因为你觉得人族太弱，不值得你效忠。那好——我就让你活着，让你亲眼看着人族站起来。让你每一天都知道，你选错了。"
苍梧的嘴唇颤抖，说不出话来。
陈玄直起身，转向众人："从今日起，苍梧逐出人道宗，永不录用。剥去修为，收回功法。任何人族领地，不得收留此人。"
```

**剧情内容**：

- 小石执行命令，以灵力封印苍梧的修为
- 苍梧被押出大殿，弟子们自动让开一条路，无人看他一眼
- 陈玄望着苍梧的背影，眼中闪过一丝复杂——苍梧曾经也是与他一同创建人道宗的人

#### 结尾（约500字）

苍梧被驱逐出人道宗后，在荒野中流浪了数日。修为被封，他连普通的野兽都无法对抗，只能靠野果和溪水苟活。

第五日，他终于找到了巫族的营地。

"我来了。"苍梧站在营地门口，衣衫褴褛，形容枯槁，"我有人道宗的情报，我可以帮你们——"

巫族的守卫上下打量着他，眼中满是轻蔑。

"一个背叛自己族人的东西，也配跟我们谈条件？"

苍梧被带进了营地，但不是作为盟友，而是作为一条丧家之犬。巫族首领冷冷地看了他一眼，对左右说道："关起来。有用的时候再拿出来。"

苍梧被推进一个阴暗的帐篷，门口站着两个巫族守卫。他坐在冰冷的地上，终于明白了一件事——

在巫族眼中，他连一条狗都不如。一条背叛主人的狗，谁会信任？

他闭上眼睛，脑海中浮现出陈玄在大殿中说的那句话——

"让你每一天都知道，你选错了。"

秋风从帐篷的缝隙中灌入，带着刺骨的寒意。苍梧缩了缩身子，这是他第一次，真正感到了冷。

**伏笔F072埋设**：苍梧投靠巫族但被软禁监视，巫族只把他当作一个可利用的棋子，从不信任。苍梧的处境为后续可能的命运转折留下空间。承接伏笔F065。
