### 三章闭环 #1：巫妖大战升级（第1-3章）

#### 第1章：大战前夕

**章节信息**：
- 字数：2500-3000字
- 节奏定位：冲突引入
- 核心任务：建立巫妖大战的背景，人族面临危机

**剧情大纲**：

```
【开篇】
巫妖大战进入白热化。
洪荒大地沦为战场，到处都是战火和废墟。

【中段】
陈玄站在人道宗的城墙上，看着远方的天际。
那里，巫族和妖族的大军正在激战。
战斗的余波震得大地颤抖，天空都被染成了血红色。

【高潮】
玄黄老人的声音响起：
"巫妖大战……终于到了最激烈的时刻。"
"这意味着什么？"
"意味着……人族将被卷入漩涡。"
```

**爽点设计**：
- 小爽点：陈玄的战略眼光（分析局势）
- 章末钩子：人族将被卷入漩涡

**场景渲染**（古风意境渲染师·战场框架）：
```
【战前寂静】
黄昏。天边的残阳如血，将整个人道宗染成暗红色。
风停了。连平日里聒噪的乌鸦都噤了声。

【危险信号】
远方的天际，两道巨大的身影正在激战。
一道是巫族的祖巫，身高万丈，浑身散发着蛮荒之气。
一道是妖族的妖圣，化作本体，是一只遮天蔽日的金翅大鹏。
它们的战斗，震得大地龟裂，山河变色。

【意境升华】
陈玄看着那两道身影，心中涌起一股深深的无力感。
这就是巫妖大战——
两大霸主的战争，而人族，只是蝼蚁。
```
